〇はじめに
kanji_data.xlsxのデータからランダムに10問選んで体裁をいい感じに整えたテキストファイルを表示してくれるアプリです。出てきたファイルを印刷して、答えを折り返して使ってください。


〇各ファイルの説明
・kanji_data.xlsx
編集すると漢字データを追加できます。
列さえ弄らなければ行は増やしても減らしても正常に動作します。
タイトルは変えないでください。

・kanji_test.exe
これをクリックすると実行できます。
出てきたファイルを印刷してください。

他は弄る必要ないです。


〇注意事項
・問題の長さは20語くらいが限界なので、あまりに長い例文を入れるとうまくいきません。
・フォントサイズは14が適度です。印刷して行が変な感じになってしまったら、フォントサイズが大きすぎる場合があるのでメモ帳の「書式→フォント」で変えてください。

・問題数を変える
